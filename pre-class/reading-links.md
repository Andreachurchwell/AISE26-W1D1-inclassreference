*   **"The Art of Problem Decomposition" (10 min)**

    *   Key Question: How is decomposition like planning a big event?

*   **"Why Professional Developers Use IDEs" (10 min)**
    *   URL: https://code.visualstudio.com/docs/python/python-tutorial#_prerequisites
    *   Focus: Benefits of VS Code for Python
    *   Key Question: What features help developers be more productive?
