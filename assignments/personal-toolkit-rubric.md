## Submission Requirements (Breakdown):

**Code Quality (40 points)**
*   Functions run without errors
*   Proper Python syntax throughout
*   Logical organization and flow
*   No code repetition (DRY principle)

**Documentation (20 points)**
*   Complete docstrings for all functions
*   Clear parameter descriptions
*   Example usage in docstrings
*   Header with author information

**Functionality (25 points)**
*   Minimum 5 working functions
*   Functions solve real problems
*   Appropriate use of parameters and returns
*   At least one function uses another

**Professional Practices (15 points)**
*   Type hints for parameters and returns
*   Error handling where appropriate
*   Descriptive variable names
*   Main function organizing demonstrations
